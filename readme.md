# POC de Automação - SauceDemo (Playwright + JS)

## Objetivo
Automatizar o fluxo completo de login, navegação, validação de produto e finalização de compra no site [SauceDemo](https://www.saucedemo.com/).

## Ferramentas
- Playwright
- Node.js / JavaScript
- Page Object Model (POM)
- Relatório HTML integrado
